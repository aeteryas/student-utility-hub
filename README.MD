# Student Utility Hub ⚡

A futuristic multi-page productivity website built using **HTML5, CSS3, and JavaScript** for students who want all essential academic tools in one place.

Designed with a sleek **black + neon green cyber aesthetic**, glowing UI elements, and interactive dashboard layout.

---

## Overview

Student Utility Hub combines useful daily student tools such as task management, notes, timers, attendance tracking, expense calculation, and motivation — all inside one centralized web platform.

## 📌 Tools Included

### ✅ Todo Manager
- Add tasks
- Delete tasks
- Mark completed tasks
- Stores tasks using Local Storage

### 📝 Notes App
- Create personal notes
- Save notes in browser
- Delete notes anytime

### ⏱ Study Timer
- Pomodoro productivity timer
- Start / Pause / Reset buttons
- Study + break cycle support

### 📊 Attendance Tracker
- Enter attended classes
- Enter total lectures
- Calculates percentage instantly
- Useful shortage checker

### ₹ Expense Calculator
- Add expenses
- Calculates total amount
- Useful for students managing monthly budgets

### ⚡ Motivation Page
- Motivational quotes
- Positive productivity vibes
